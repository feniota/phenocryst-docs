# Next Steps

## If You Are a Player

You should now have the Phanerite launcher downloaded. Just open it, enter the ~~magic~~ code given by your server owner, log in with the account you registered on the link your server owner provided, optionally set your skin, and wait a bit for the game to start.

## If You Are a Server Owner

Aphanite needs configuration before it can do its job. Head over to [Aphanite's documentation](/aphanite/) for details.

## Found a Bug / Have a Suggestion / Want to See the Source / Want to Contribute?

Both Aphanite and Phanerite are open source software. Their repositories:

- Aphanite: https://github.com/feniota/aphanite
- Phanerite: https://github.com/feniota/phanerite

If you find a bug or have a suggestion, feel free to open an Issue!

If you'd like to contribute, Pull Requests are very welcome! (Please note that for code entirely generated by AI, we may take a stricter stance during review.)
