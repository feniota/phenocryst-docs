# Yggdrasil API

Aphanite 在 `<aphanite_base_url>/api/yggdrasil` 下实现了 Yggdrasil 服务。通过使用 authlib-injector，Aphanite 可以代替 Mojang 服务器执行玩家信息验证和皮肤下发等功能。

由于这是一个相对广为人知的 API，这里就不单独介绍各端点了，可以参考下面的几个文档链接。
